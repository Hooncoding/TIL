파이썬 데이터 타입: 문자형
=========================

  우리는 앞으로 파이썬을 주로 데이터 분석 툴로 이용할 것이다.  
이 과정에서 문자형 데이터를 가장 많이 다룰 것 이므로 문자형 선언과 관련 함수를 잘 이해해야 한다. 

문자형 변수 선언하기
-------------------

**문자열 선언**

```python

작은 따옴표로 선언하기

>>> a = 'abc'
>>> a
'abc'
----------------------

큰 따옴표로 선언하기

>>> a = "abc"
>>> a 
'abc'
```

**문자열 내 작은/큰 따옴표 삽입**
```python
작은 따옴표 삽입

>>> a = "Dylan's repository"
>>> a
"Dylan's repository"
-----------------------

큰 따옴표 삽입

>>> a = 'he said "hi"'
>>> a
'he said "hi"'
```

**문자열 여러 줄 선언하기***

```python
>>> a = ''' hi
... my 
... name
... is dylan '''
>>> a
' hi\nmy\nname\nis dylan '
```
\n 은 줄바꿈을 의미한다.

문자열 활용하기
---------------

**문자열 합치기**

```python
>>> a = "ab"
>>> b = "cd"
>>> a + b
'abcd'
```

이전 시간에 + 연산자는 add 함수를 실행한다고 했다. add 함수는 파라미터 두개를 받는다.  
이때 파라미터에 숫자가 들어가면 두 숫자를 합한 숫자를 반환하고  
파라미터가 문자열일 경우 두 문자열을 연결한 문자열을 반환한다: add 함수 OverLoading.  

**문자열 반복하기**

```python
>>> a = 'hi'
>>> a * 2
'hihi'
```
곱 연산자도 합 연산자와 같은 메커니즘(__mul__(self, other) 함수 실행)으로 동작된다.

**문자열 인덱싱과 슬라이싱**

'hello'라는 문자열이 있다고 하자.  
여기서 문자열이란 정확히는 '문자의 리스트'이다.  
즉 0, 1, 2 ,3, 4(인덱스)를 부여받은 각 문자 h, e, l, l, o를 가지고 있는 목록인 것이다.  
정확히는 str이 tuple을 상속받고 tuple이 list를 상속받기 때문에 str은 list의 속성을 갖는다.  

중요한 점은 우리가 이 인덱스를 활용해서 문자열을 내 문자를 찾거나 자를 수 있다는 것이다.

**인덱싱: 인덱스로 문자열 내 문자 가져오기**

```python
앞에서 부터 인덱싱
>>> a = 'hello'
>>> a[0]
'h'
>>> a[4]
'o'
---------------------

뒤에서 부터 인덱싱
주의: 뒤에서 부터 인덱싱 시 -1 부터 시작

>>> a[-1]
'o'
>>> a[-5]
'h'
```

**슬라이싱: 문자 잘라내기**

```python
문자열 선언 

>>> a = a.jpg
>>> b = bb.png

확장자만 잘라내기

>>> a[-3 : ]
jpg              
>>> b[-3 : ]
png
------------------

파일명만 잘라내기

>>> a[ : -4]
a
>>> b[ : -4]
bb
```

**문자열 포매팅**

문자열 포매팅이란 문자열의 형식을 지정하는 것이다.  
문자열 내 %s, %d 등을 삽입 후 문자열 뒤에 % 삽입할 값을 입력해주면 된다.  
여기서 삽입할 값이 문자열이면 %s, 숫자면 %d을 사용한다.  
예를 들어, 1시간마다 현재 온도를 출력하는 프로그램을 만들고자 한다.

```python
문자열 내 문자열 포매팅

>>> a = '현재 온도는 %s도 입니다.' % '이십'
>>> a
'현재 온도는 이십도 입니다.'
---------------------------

문자열 내 숫자 포매팅

>>> a = '현재 온도는 %d도 입니다.' % 18
>>> a
'현재 온도는 18도 입니다.'

---------------------------

문자열 내 변수 포매팅

>>> temp = 18
>>> a = '현재 온도는 %d도 입니다.' % temp
>>> a
'현재 온도는 18도 입니다.'
 
주의할 점: 변수가 문자열인데 포매팅을 %d로 하면 에러.
다만, 변수가 숫자인데 포매팅을 %s로 하는건 가능 (숫자 -> 문자 자동 형변환)

----------------------------

문자열 내 다수 값 포매팅

>>> a = "현재 온도는 %d도 이고, 날씨는 %s 입니다."%(18, '맑음')
>>> a
"현재 온도는 18도 이고, 날씨는 맑음 입니다."

>>> temp = 18
>>> weather = '맑음'
>>> a = "현재 온도는 %d도 이고, 날씨는 %s 입니다."%(temp, weather)
>>> a
"현재 온도는 18도 이고, 날씨는 맑음 입니다."

-----------------------------

포매팅 사용한 문자열 내 '%' 문자 사용하기

>>> a = "현재 온도는 %d도 이고, 습도는 %d%% 입니다."%(18, 90)
```

**문자열 정렬 및 공백 넣기**
```python

문자열 10 칸 생성 -> 오른쪽 정렬 -> 남는 칸 공백 넣기

>>> a = "%10s" % 'hi'
>>> a
'        hi'
------------------------------

문자열 5 칸 생성 -> 왼쪽 정렬 -> 남는 칸 공백 넣기

>>> a = "%-5s" % 'hi'
>>> a
'hi   '

```

**소수점 처리**
```python

소수점 n자리에서 절사 (%0.nf)

>>> a = "%0.4f" % 3.141592
>>> a
'3.1415'
---------------------------

소수점 n자리 절사 및 정렬

>>> a = "%-10.4f" % 3.141592
>>> a
'3.1415    '

```

문자열 관련 함수
----------------

**문자 세기: count()**

```python
>>> a = 'abb'
>>> a.count('b')
2
>>> a.count('bb')
1
```

**문자 위치 반환: find(), index()**

```python

find 이용하여 찾기

>>> a = 'abbccc'
>>> a.find('b')
1
>>> a.find('k')
-1

문자열 내 b가 가장 먼저 나오는 index 반환.
문자열 내 찾는 문자가 없을경우 -1 반환.
-------------------

index 이용하여 찾기

>>> a = 'abbccc'
>>> a.index('c')
3
>>> a.index('k')
에러

index는 find와 다르게 찾는 문자가 없으면 에러.
```

**문자열 삽입: join()**

```python

>>> a = ' , '
>>> b = '와'
>>> a.join('abcd')
'a , b , c , d'
>>>b.join('abcd')
'a와b와c와d'

```

**공백 지우기: strip()**

```python
>>> a = '   hi   '
>>> a.lstrip()
'hi   '
>>> a.rstrip()
'   hi'
>>> a.strip()
'hi'
```

**문자열 바꾸기: replace()**

```python
>>> a = 'Life is too short'
>>> a.replace('Life', 'Your Leg')
'Your Leg is too short'
```

**문자열 나누기: split()**
```python

공백 기준으로 나누기 : split()

>>> a = 'Life is too short'
>>> b = a.split()
>>> b
['Life', 'is', 'too', 'short']

이때 b는 list형 변수이다.
-----------------------------

특정 문자로 나누기 : split('문자')

>>> a = 'a,b,c,d'
>>> b = a.split(,)
>>> b
[a, b, c, d]

```

**함수로 포매팅하기: format()**
```python

숫자 포매팅

>>> a = 'I eat {0} apples'
>>> a.format(3)
'I eat 3 apples'
--------------

문자 포매팅

>>> a = 'I eat {0} apples'
>>> a.format('three')
'I eat three apples'
--------------

변수 포매팅

>>> number = 3
>>> a = 'I eat {0} apples'
>>> a.format(number)
'I eat 3 apples'
----------------

다수의 값 포매팅

>>> number = 3
>>> number2 = 'two'
>>> a = 'I eat {0} apples, {1} peaches'
>>> a.format(number, number2)
'I eat 3 apples, two peaches'
------------------

변수명 직접 삽입하여 포매팅

>>> a = 'I eat {num} apples, {num2} peaches'
>>> a.format(num=3, num2='two')
'I eat 3 apples, two peaches'
--------------------

포맷하고 정렬하기

>>> a = '{0:!<10} dylan"
>>> a.format('hi')
'hi!!!!!!!! dylan'

0: 포맷 인덱스 (값 받을 자리)
':' : 구분자
< : 왼쪽 정렬 하시오(오른쪽정렬 '>', 가운데 정렬 '^')
! : 남은 공간 !로 채우시오 (입력 x 시 공백)
10: 10자리 공간 만드시오
------------------------

소수점 처리

>>> a = '{0:0.4f}'
>>> a.format(3.14159)
'3.1416'

format 함수는 소수점 절사할 시 마지막 자리에서 반올림 한다.
