파이썬 데이터 타입: 리스트 형
===========================
파이썬에서는 리스트 형이라는 데이터 타입이 존재한다.  
자바에서의 배열과 비슷하지만 자바의 Array, List와는 상이하다.  

**자바 배열과의 차이점**

+ 자바 배열은 선언시 배열이 저장할 데이터 타입을 입력해야 한다.  
  > 파이썬은 그렇지 않다!
+ 자바 배열은 선언한 데이터 타입의 변수나 값만 대입할 수 있다.  
  > 파이썬은 그렇지 않다!

자바 배열의 특징

```java
int[] intarr = {1, 2, 3};

int[0] = "일"; => 에러!!!!
```

파이썬 List의 특징

```python
>>> a = [1, 이, 3.5]
>>> a
[1, '이', 3.5]

```

**자바 List와의 차이점**

+ 자바 List는 index가 존재하지 않는다.
  > 파이썬은 그렇지 않다!
+ 자바 List를 반복문 처리 할 시 Iterater를 사용해야 한다
  > 파이썬은 그렇지 않다!

List 형 변수 선언하기
----------------------
List 형 변수 내 component들은 데이터 타입에 상관없이 삽입할 수 있다.  
파이썬에서는 기본 타입이 없기 때문에 List 내 들어오는 값들이 모두 참조형으로 동일하다.  
정확히는, List 안에 들어갈 수 있는 값은 4바이트 크기의 정수(참조형)면 되기 때문에 모든 값이 들어갈 수 있다.


```python
 >>> a = []
 >>> b = [1,2,3,4]
 >>> c = ['a', 'b', 'c']
 >>> d = [1, 2, 'a', 'b']
 >>> e = [1, 2, ['a', 'b']]
 ```
 
 List 인덱싱
 ----------------------
 
 ```python
 인덱스로 값 찾기
 
 >>> a = [1,2,3]
 >>> a[0]
 3

인덱스로 연산하기

>>> a = [1,2,3]
>>> a[0] + a[1]
3

인덱스로 값 찾기 (2차원 List Case)

>>> a = [1,2,'hi',['a','b','c']]
>>> a[3][1]
'b'

>>> a[2][1]
'i'

인덱스로 값 수정하기

>>> a = [1,2,3]

[1,2,4]

```

List 슬라이싱
--------------
```python
슬라이싱으로 값 찾기

>>> a = [1,2,3,4,5]
>>> a[0:2]
[1,2]

슬라이싱으로 값 찾기(2차원 List case)

>>> a = [1, 2, ['a', 'b', 'c']]
>>> a[2][:2]
['a', 'b']

슬라이싱으로 수정하기

>>> a = [1,2,3]
>>> a[1:2] = ['a','b']
>>> a
[1,'a','b',3]

>>> a = [1, 'a', 'b', 'c', 2]
>>> a[1:3] = []
>>> a
[1, 'c', 2]

*** 주의할 점 ***

>>> a = [1,2,3]
>>> a[1] = [1,2]
>>> a
[1,[1,2],3]

>>> a = [1,2,3]
>>> a[1:2] = [1,2]
>>> a
[1,1,2,3]

인덱싱과 슬라이싱으로 값 수정할 경우 차이점: 슬라이싱으로 List, Tuple, Str 삽입 시 컴포넌트들 자체가 리스트에 들어감

```

List의 연산자 활용
-------------------

**List 합치기(+)**
```python
>>> a = [1,2,3]
>>> b = [4,5,'hi']
>>> a + b
[1,2,3,4,5,'hi']

>>> a[2] + b[1]
```

```python

```

del 함수 사용하여 List 값 삭제하기
----------------------------------
del 함수는 object 자체를 삭제하는 파이썬 내장 함수이다.  
정확히는 메모리상에 해당 object가 차지하는 공간을 삭제해버리는 것이다.  
Java에서는 메모리 공간 삭제를 직접적으로 할 수 없었다. => 파이썬은 가능하다!  
del 함수를 사용하여 List[n] 값을 삭제하면 List 내 n번 인덱스에 있는 값(주소)이 가리키는 object를 지워버린다.  
그 결과 n번 인덱스의 값(주소)이 가리키는 Object가 없기 때문에 n번 인덱스 내 값(주소)는 사라지고 n+1번이 밀려서 n번 인덱스 값이 된다.

```python
>>> a = [1,2,3]
>>> del a[1]
>>> a

>>> a = [1,2,3,4]
>>> del a[1:3]
>>> a
[1,4]
```

List 클래스 내 메소드
----------------------

**List 값 추가: append**

값을 List의 맨 마지막에 추가하는 함수

```python
>>> a = [1,2,3]
>>> a.append(4)
>>> a.append('hi')
>>> a.append([1,2])
>>> a
[1,2,3,4,'hi',[1,2]]
```

**List 값 추가: extend**

extend 함수를 사용하여 List 내 List(Tuple, Str)를 추가 시 List가 아닌 List 내 요소들이 추가된다.

```python
>>> a = [1,2,3]
>>> a.extend([1,2])
>>> a
[1,2,3,1,2]
```
**List 정렬**

sort는 List 내 값들을 '사전식 기준'으로 정렬해주는 역할을 한다.

```python
>>> a = [1,4,3,2]
>>> b = ['a','c','b']
>>> c = [1,4,3,'a','c','b']
>>> a.sort()
>>> b.sort()
>>> c.sort()
c.sort()는 에러 숫자와 문자간 사전식 기준은 정의되지 않음

>>> a
[1,2,3,4]

>>> b
['a','b','c']
```
**List 뒤집기**
reverse 함수는 List내 값들을 역순으로 재배치 (정렬 기능은 X)

```python
>>> a = ['a', 'c', 'b']
>>> a.reverse()
>>> a
['b', 'c', 'a']
```

**List 값 위치 반환**

index() 함수는 특정 값의 index를 찾고 싶을 때 사용한다.

```python
>>> a = [1,2,3]
>>> a.index(3)
2

>>> a.index(100)
에러 !!!

```

**List 위치에 값 추가**

insert 함수는 append, extend와 다르게 원하는 위치에 해당 값을 삽입할 수 있다.

```python
>>> a = [1,2,3]
>>> a.insert(1,5)
>>> a
[1,5,2,3]

>>> a.insert(3,6)
>>> a
[1,5,2,6,3]
```
**List 값 삭제**

remove() 함수는 List내 해당 값을 찾아 지운다.  
이 때, 해당 값이 여러개면 가장 빠른 값 (index가 제일 작은 값)을 지운다.  

```python
>>> a = [1,2,3,1,2,3]
>>> a.remove(3)
>>> a
[1,2,1,2,3,]
```
**List 값 추출**

pop(index) 함수는 List 내 index에 해당하는 값을 추출하고, List에서 해당 값을 삭제하는 함수이다.  


```python
>>> a = [1,2,3]
>>> b = a.pop(2)
>>> a
[1,2]
>>> b
3

>>> c = a.pop(1)
>>> a
[1]
>>> c
2
```

**List 내 값 갯수 세기**

count(값) 함수는 List내 해당 값이 몇개나 있는지 찾는 함수이다.  

```python
>>> a = [1,2,3,3]
>>> a.count(3)
2
```
