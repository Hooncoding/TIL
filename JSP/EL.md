EL (Expression Language)
========================

What is EL?
-----------
el이란 표현식 또는 액션 태그를 대신해서 값을 **출력하는 언어**이다.
EL을 통해 우리는 복잡한 표현식, 스크립틀릿, 태그들을 단순화 할 수 있다.

EL을 사용하면 편한 이유
------------------
아래의 세 코드는 모두 같은 결과 화면을 출력한다.
**자바 코드로 웹에서 hi 출력하기**
```
PrintWriter out = response.getWriter();
            out.print("hi");
```
**스크립틀릿(표현식)**
```
<%= hi %>
```

**EL로 출력하기**
```
${hi}
```

어떤 코드가 더 간단한가? 판단은 당신에게 맡긴다.
